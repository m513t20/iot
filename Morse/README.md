# Морзе

 Реалезован режим через пользовательский ввод . -, его можно расширить до автоматического режима. 
```
Генератор ────> Serial ────> Декодер
  ↑                                    ↓
Arduino #1                      Arduino #2 (7seg)
.- ... -.                     A S N
```
