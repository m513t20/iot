# RangeCalibration: Автоматизированная калибровка ИК-дальномера

## Описание проекта
Данный проект реализует систему автоматизированной калибровки аналогового инфракрасного (ИК) дальномера, используя ультразвуковой сенсор (HC-SR04) в качестве эталонного источника данных.

Цель работы — построить математическую модель (полиномиальную регрессию), которая позволяет преобразовывать нелинейные показания `analogRead` с ИК-сенсора в точное расстояние в сантиметрах.

### Особенности реализации
*   **Сбор данных:** Микроконтроллер считывает данные одновременно с двух датчиков и передает их на ПК.
*   **Валидация выборки:** Python-скрипт анализирует поступающие данные в реальном времени. Сбор считается завершенным только при соблюдении двух условий:
    1.  Покрытие целевого диапазона (например, 10-30 см) составляет не менее **90%**.
    2.  Максимальный разрыв (пропуск) между измерениями не превышает **2 см**.
*   **Обратная связь:** Микроконтроллер сигнализирует светодиодом о процессе сбора и об успешном завершении калибровки.
*   **Математическое моделирование:** На основе собранных данных строятся полиномы 3-й, 5-й и 7-й степеней для нахождения наилучшей аппроксимации.

## Аппаратное обеспечение
Для реализации проекта используется следующая схема:
1.  **Микроконтроллер:** Arduino Uno / Nano (или аналог).
2.  **Эталонный датчик:** Ультразвуковой дальномер HC-SR04.
    *   Trig -> Pin 5
    *   Echo -> Pin 4
3.  **Калибруемый датчик:** ИК-дальномер (Sharp или аналог).
    *   Output -> Pin A0
4.  **Индикация:** Встроенный светодиод (Pin 13).

## Программная реализация

### 1. Микроконтроллер (C++)
Прошивка работает как конечный автомат, управляемый командами по Serial порту:
*   Ожидает команду `START min_dist max_dist` для начала передачи сырых данных.
*   В цикле отправляет пару значений `дистанция_уз сигнал_ик`.
*   При получении команды `STOP` переходит в режим завершения, уведомляя пользователя миганием светодиода.

### 2. Обработка данных (Python)
Скрипт на стороне ПК выполняет основные вычисления:
1.  **Фильтрация и накопление:** Считывает поток данных, отбрасывая шум.
2.  **Проверка покрытия (Coverage Check):** Функция `check_coverage` сортирует уникальные измеренные дистанции и вычисляет процент покрытия диапазона и максимальный шаг между точками.
3.  **Построение модели:** Используется метод наименьших квадратов (`numpy.polyfit`) для нахождения коэффициентов полинома, описывающего зависимость $Distance = f(AnalogSignal)$.

## Инструкция по запуску

### Предварительные требования
Установите необходимые библиотеки Python:
```bash
pip install pyserial numpy matplotlib
```

### Запуск
1.  Соберите схему согласно разделу "Аппаратное обеспечение".
2.  Загрузите код из папки `arduino` в микроконтроллер.
3.  Откройте Python-скрипт и настройте порт:
    ```python
    PORT = 'COM3' # Для Windows
    # PORT = '/dev/ttyUSB0' # Для Linux/Mac
    ```
4.  Запустите скрипт:
    ```bash
    python calibration.py
    ```
5.  **Процесс калибровки:**
    *   Скрипт отправит команду старта. Светодиод на плате загорится.
    *   Плавно перемещайте препятствие (лист бумаги или руку) перед сенсорами в заданном диапазоне (по умолчанию 10-30 см).
    *   Старайтесь двигать объект медленно, чтобы заполнить все "пробелы" в дистанции.
    *   Как только данных будет достаточно (90% покрытия), программа автоматически остановит сбор, светодиод начнет мигать, и на экране появится график.

## Результаты
По завершении работы программы отображается график зависимости расстояния от показаний АЦП. На графике представлены:
*   **Точки:** Реальные измерения (сырые данные).
*   **Кривые:** Аппроксимирующие полиномы разных степеней (Красный - 3 ст, Синий - 5 ст, Зеленый - 7 ст).

Выбранная модель позволяет переводить условные единицы (0-1023) в сантиметры с высокой точностью в пределах откалиброванного диапазона.
